基于 Node.js 的高性能 API 网关。<br />

<a name="2tCdT"></a>
## 特性

- 丰富的内置插件，可以满足大多数网关功能需求；
- 简单方便地通过 Node.js 开发自定义插件；
- 网关配置热加载，永不停机；
- 高性能，8 核达到 50000 QPS；
- 云原生，无限水平扩容；
